Code Reviews 

Code reviews will be performed manually as well as systematically using Linting tools such as Sonarqube, CodeScan and/or PMD.  The following points are meant to provide the guidance being enforced by the  DTC during code reviews.  Some points, marked as required, will result in the rejection of a given code branch and will require remediation before it can be promoted to upper environments.  Other points will be enforced less rigidly and could end up in an exception being logged for future remediation or simply logged as tech debt.
